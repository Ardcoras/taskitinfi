Taskitin.fi data protection

Data controller

The data controller is Ardcoras oy. The contact person is Jyri-Petteri Paloposki, info@ardcoras.fi.

Purpose and basis for processing personal data

Data processing is based on the data subject's consent.

The data is not used for automated decision-making or profiling.

Data content

  • Username
  • Email address
  • Creation date of the account and date of last login
  • Information related to the Stripe payments
  • User preferences
  • Information stored in the service by the user regarding tasks, their classification, projects etc.

Regular sources of data

Personal data is collected from the data subjects themselves via the service.

Regular data disclosure and transferring data outside the EU or EEA

Data is not regularly disclosed to third parties. Data is not transferred outside the EU or EEA.

Register security principles

Data is handled with care and the data is protected. Data is only handled with those who need access and they're only accessed using a secure connection with a personal user account.

Data subject rights

The data subject has the right to access their own data. They don't have the right to rectify data because all the data is entered by the data subject themselves. The request for data must be sent to the data controller.

The data subject has the right to request erasing the information concerning themselves.
